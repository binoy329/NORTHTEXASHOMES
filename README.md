## Preview on your computer

Open `index.html` in a web browser.

## Add Binoy's photo

1. Save the headshot inside this folder as `binoy-thomas.jpg`.
2. In `index.html`, replace each photo placeholder `<div>` with:

```html
<img src="binoy-thomas.jpg" alt="Binoy Thomas, North Dallas real estate professional">
```

3. Add an appropriate CSS class or use:
```css
img { width: 100%; height: 100%; object-fit: cover; border-radius: 16px; }
```

## Publish free with Netlify Drop

1. Unzip the website.
2. Open Netlify Drop in your browser.
3. Drag the entire `elegant-homes-website` folder into the upload area.
4. Netlify will create a free public web address.
5. You can later connect a custom domain.

## Important notes
